## 在线文档

## 目录结构

- `cigbutt/` — 烟蒂股分析 Prompt（多版本迭代）
- `turtle_framework/` — 龟龟投资策略 Prompt（多版本迭代）
- `reports/` — 基于以上 Prompt 生成的个股分析报告
